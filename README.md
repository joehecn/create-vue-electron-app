# create-vue-electron-app
create an app by vue3 and electron quickly!

``` bash
$ npm init vue-electron-app <project-name>
# or
$ npx create-vue-electron-app <project-name>

$ cd <project-name>
$ npm install
$ npm run dev
$ npm run build
```

If using Yarn:

``` bash
$ yarn create vue-electron-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```
